# JavaFX Chessboard

## Overview
This project is a JavaFX-based chessboard application that visually displays a standard 8x8 chessboard with chess pieces.

## Features
- 8x8 grid using JavaFX GridPane
- Alternating board colors
- Chess piece images (black & white)
- Clean UI layout

## Technologies Used
- Java
- JavaFX
- Maven

## Skills Demonstrated
- GUI development with JavaFX
- Layout management using GridPane
- Object-oriented programming
- Project structuring with Maven

## How to Run
1. Clone the repository
2. Open in IntelliJ or any Java IDE
3. Run `ChessBoard.java`
